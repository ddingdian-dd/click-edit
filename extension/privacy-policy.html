Click-Edit Privacy Policy

Last updated: 2026-06-15

Overview

Click-Edit is a visual page editor Chrome extension that allows users to select and modify page elements in real-time. We are committed to protecting your privacy.

Data Collection

Click-Edit does not collect, transmit, or store any personal data. Specifically:

  • No browsing history is collected
  • No personal information is gathered
  • No data is sent to external servers
  • No cookies are set by the extension
  • No analytics or tracking is performed

Local Storage

Click-Edit stores your editing records exclusively in the browser's local storage (localStorage). This data:

  • Never leaves your device
  • Is only used to re-apply your edits when you revisit a page
  • Can be cleared at any time via your browser settings

Permissions Explanation

  • activeTab: Required to inject the editor into the current tab when activated by the user
  • scripting: Required to run the visual editor on the page
  • storage: Used to remember whether the editor is active on a given tab
  • Host permissions (all URLs): Required so the editor can work on any website the user chooses to edit

Third-Party Services

Click-Edit does not integrate with any third-party services or APIs.

Changes to This Policy

Any updates to this privacy policy will be reflected on this page with an updated date.
